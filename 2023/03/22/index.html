hexo 常用指令

  • hexo g : 生成靜態文件，會在 Hexo 項目的根目錄下生成一個 public 目錄，用於存放生成的靜態文件。

  • hexo s : 用於啟動本地服務器。

  • hexo d : ：將生成的靜態文件部署到遠程服務器或雲端平台，如 github。

  • hexo new : 創建一篇新的 markdown 文章。

  • hexo clean : 清除 Hexo 項目中生成的靜態文件和緩存文件，以便重新生成靜態文件。

  • hexo d -g : 結合 hexo g 以及 hexo d，將生成的靜態文件推送到 GitHub Pages 上。
